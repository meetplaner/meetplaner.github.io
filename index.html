// USER GUIDE & TIPS

MeetAlarm 완벽 활용 가이드

처음 시작하는 관리자와 초대받은 팀원을 위한 상세한 단계별 사용 팁입니다. 어떻게 활용해야 가장 효율적인지 사례와 함께 설명해 드립니다.

1. 🏢 관리자 초기 셋업: 회사 생성과 초대코드 배포

MeetAlarm의 가장 큰 특징은 복잡한 이메일 회원가입 없이, '초대코드(Invite Code)' 기반으로 조직을 빠르고 안전하게 세팅할 수 있다는 점입니다.

🖼️ [ 앱 실행 화면 - '새 회사 등록' 탭에서 회사명을 입력하고 초대코드를 발급받는 모습 ]

✅ 초기 셋업 TIP

  • 조직 세분화: 예를 들어 "본사", "베트남 신규 플랜트 TFT" 처럼 협업 목적에 맞게 여러 회사를 분리해서 생성하세요. 각 회사의 데이터(미팅, 일정, 직원 목록)는 서버 레벨에서 완벽하게 격리됩니다.
  • 초대코드의 중요성: 앱 첫 화면에서 '새 회사'를 만들면 6자리의 고유 초대코드(예: AB1C23)가 발급됩니다. 이 코드는 외부인이 그룹에 들어오는 것을 막는 가장 중요한 보안키(Master Key)입니다. 관리자는 이 코드를 복사하여 사내 메신저나 슬랙을 통해 팀원들에게 전달해야 합니다.
  • 부서 및 조직도 구성: 회사를 만들고 진입했다면, 가장 먼저 좌측 메뉴의 [구성원 관리]로 이동하여 '영업팀', '생산팀', '품질관리팀' 등 부서를 생성하세요. 부서별로 지정한 색상은 캘린더에 그대로 반영되어 뛰어난 가독성을 제공합니다.

💡 보안 팁: 만약 초대코드가 외부로 유출되었다면, [연동 설정] 메뉴의 관리자 전용 탭에서 '새 코드 발급'을 눌러 기존 코드를 폐기하고 즉시 새로운 코드로 교체할 수 있습니다.

2. 👥 팀원 합류 및 스마트한 앱 활용법

관리자에게 초대코드를 전달받은 실무자들은 단 10초면 팀에 합류할 수 있습니다.

🖼️ [ 앱 실행 화면 - '초대코드' 탭에서 코드를 입력하고, 자신의 이름과 비밀번호를 설정하는 모습 ]

✅ 실무자 활용 TIP

  • 간편한 합류: 첫 화면에서 [초대코드로 가입]을 선택하고 전달받은 코드를 입력하세요. 본인의 이름과 4자리 이상의 비밀번호만 설정하면 가입 절차가 모두 끝납니다.
  • 앱 설치 효과 내기 (PWA): 매번 웹 브라우저를 열 필요가 없습니다. 스마트폰 사파리나 크롬 브라우저 하단의 공유 버튼을 눌러 [홈 화면에 추가]를 선택하세요. 바탕화면에 MeetAlarm 아이콘이 생기며, 일반 네이티브 앱과 동일한 전체화면으로 쾌적하게 구동됩니다.
  • Zoom 회의 참여: 스케줄러에 등록된 미팅 카드를 보면 [📹 입장 →] 버튼이 활성화되어 있습니다. 회의 시작 5분 전, 단톡방에서 화상 회의 링크를 찾아 스크롤을 올릴 필요 없이 여기서 클릭 한 번으로 즉시 회의실에 입장하세요.

3. 🔄 멀티 테넌트: 이중 점검과 다중 계정 전환

N잡러, 프리랜서, 혹은 여러 프로젝트 팀에 동시에 속해 있는 사용자를 위해 MeetAlarm은 강력한 멀티 회사 전환 기능을 제공합니다.

✅ 다중 계정 동기화 TIP

  • 데이터 유실 걱정 끝: 스마트폰 브라우저가 임의로 로컬 캐시를 삭제하여 첫 화면이 나타났더라도 당황하지 마세요. 앱 내부의 [회사 전환] 메뉴로 들어가 [서버 데이터와 내 목록 동기화]를 사용하면 됩니다. 사용 중인 대표 이름과 비밀번호만 입력하면 백엔드 서버에서 귀하가 속한 모든 회사의 리스트를 즉시 끌어와 복구해 줍니다.
  • 원클릭 스위칭: 회사 목록에서 [🔄 전환] 버튼만 누르면 로그인/로그아웃 과정 없이 A회사 환경에서 B회사 환경으로 즉시 넘나들 수 있습니다.

4. ⏰ 일정 관리의 꽃: 3단계 알람과 Slack 연동

업무에 집중하다 보면 회의 시간을 놓치기 일쑤입니다. MeetAlarm의 시스템은 이러한 인적 오류(Human Error)를 방지하도록 설계되었습니다.

  • 타겟형 3단계 알람: 미팅 생성 시 참석자로 지정된 인원에게만 알람이 울립니다. 회의 60분 전, 30분 전, 10분 전에 맞춰 스마트폰과 PC 브라우저에 팝업 알림을 보내줍니다.
  • Slack Incoming Webhook: [연동 설정]에서 슬랙 웹훅 URL을 한 번만 등록해 두면, 미팅이 생성되거나 알람이 울릴 때 사내 지정된 슬랙 채널로 안내 메시지가 자동 송출됩니다. 팀원들의 참석률을 100%로 끌어올리는 가장 확실한 방법입니다.

자주 묻는 질문

MeetAlarm 도입을 검토 중이신가요? 많은 분들이 궁금해하시는 핵심 질문들에 대한 답변을 준비했습니다.

아닙니다. MeetAlarm은 클라우드 기반의 최신 웹 앱(Web App)으로 설계되어 구글 플레이스토어나 앱스토어에서 무거운 앱을 다운로드할 필요가 없습니다. 모바일 기기의 브라우저나 데스크톱 브라우저에서 제공되는 접속 링크로 진입하면 즉시 모든 기능을 사용할 수 있습니다. 더욱 편리하게 사용하시려면 기기 설정에서 '홈 화면에 바로가기 추가(PWA)' 기능을 활용해 보세요.
이는 아이폰 Safari의 개인정보 보호 모드나 카카오톡 인앱 브라우저 등에서 보안을 위해 자체적으로 로컬 접속 캐시를 지워버리기 때문에 발생하는 현상입니다. 하지만 서버에는 모든 정보가 안전하게 보관되어 있습니다.

첫 화면에서 다시 초대코드로 임시 가입하여 접속하신 후, 좌측 메뉴의 [회사(그룹) 전환]에 들어가세요. 하단의 "내 모든 그룹 한 번에 불러오기" 탭에 기존 가입 이름과 비밀번호를 입력하시면 서버에서 모든 소속 회사 리스트를 가져와 1초 만에 완벽하게 원상 복구해 드립니다.
네, 가능합니다! 본사에서는 '홍길동'으로 가입하고, 외부 TFT 프로젝트 팀에서는 '기획팀 홍길동'으로 각각 다르게 가입했더라도 문제가 없습니다. 앱 내부의 [회사(그룹) 전환] 메뉴 하단에 있는 '새 회사(그룹) 연결하기' 기능을 이용하세요. 해당 회사의 초대코드, 정확한 회사명, 가입 시 사용한 이름과 비밀번호를 입력하면 현재 내 기기의 목록에 통합 연동시킬 수 있습니다.
좌측 메뉴의 [정기 미팅 관리] 기능을 이용하면 관리가 획기적으로 편해집니다. 매주 월요일 오전에 진행하는 주간 회의 등을 '정기 미팅 템플릿'으로 한 번만 등록해 두세요. 지정된 요일이 포함된 달력을 띄울 때 시스템이 자동으로 스케줄을 생성해 줍니다. 이번 주 회의 날짜만 예외적으로 화요일로 바뀌었거나 취소되었다면, 해당 일정 클릭 후 '이번 회차만 수정' 기능을 통해 유연하게 대처할 수도 있습니다.
네, 프라이버시가 보호되는 개인 일정을 완벽히 지원합니다. [개인 및 팀 일정] 메뉴에서 일정을 등록할 때 '공개 범위'를 설정할 수 있습니다.

'개인'으로 설정하면 나 자신에게만 보이며, '팀 공개'로 설정하면 같은 부서원들에게, '전사 공개'로 설정하면 회사 내 모든 인원에게 일정이 공유됩니다. 이를 통해 사적인 스케줄과 공적인 업무 스케줄을 하나의 캘린더에서 섞이지 않게 관리할 수 있습니다.
